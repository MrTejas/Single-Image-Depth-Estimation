This package includes the following files:
1) This readme file in the root path.
2) The path of depth contains full codes and test data for depth estimation. 
   Run estimate_depth.m inside on MATLAB to generate a depth map.
3) The path of functions contains supporting functions, and the path of bin contains c++ implementation of BP. 

The source code is available for research purposes.
